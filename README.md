# 情圣 QingSheng — Skill v3

中国男性社交表达与恋爱全流程私人教练。

## v3 新特性

- **用户人设系统**：基于你的真实兴趣和特长打造吸引力人设，永远不替你编故事
- **平台差异化**：覆盖微信、探探、陌陌、Soul、Bumble、青藤之恋，针对每个平台给出差异化的 profile 优化和聊天策略
- **截图自动分析**：上传截图直接开始分析，不问多余的问题
- **"不回复"判断**：不是每条消息都要回，告诉你什么时候沉默比说话更有力量
- **快捷指令**：`/regenerate` 重新生成不满意的建议，`/ask` 快速提问模式
- **emoji 融入**：回复建议自然带上表情，可以直接复制粘贴
- **语言人类化**：消除 AI 痕迹，输出像真人聊天一样自然
- **幽默度标尺**：话术标注轻松/调皮/大胆，根据你的性格选择
- **降低攻击性**：温暖幽默型取代毒舌型，调侃让人笑不让人不舒服
- **深度上下文管理**：完整对话时间线、素材存档、引领规划追踪

## v2 特性（保留）

- **多目标上下文管理**：自动识别你在聊谁，为每个女生维护独立档案
- **主动引领模式**：话题引领、情绪引领、节奏引领
- **框架意识进阶**：框架对决、情绪领导力

## 功能概述

覆盖从开场白到确立关系的完整 **七阶段推进体系**：

1. 开场破冰
2. 建立好感
3. 关系升温
4. 邀约见面
5. 约会实战
6. 亲密升级
7. 确立关系

支持上传聊天记录、朋友圈截图、头像、个人档案等素材做深度分析。

## 适用场景

- 不知道怎么和女生聊天 / 不会回消息
- 想追一个人不知道怎么开口
- 聊天冷场了怎么办
- 被已读不回了怎么办
- 暧昧期不知道怎么推进关系
- 约会前不知道说什么
- 分析对方是否对自己有意思
- 同时在和多个女生聊天，需要分别管理
- 感觉自己总是被动，想学会引领对话
- 不知道在探探/Bumble/Soul 等平台上怎么优化 profile 和开场白
- 想打造一个真实有吸引力的个人形象
- 对上一次的建议不满意，想换个风格（`/regenerate`）
- 紧急情况需要快速回答（`/ask`）
