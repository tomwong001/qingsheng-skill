### Demo 1：聊天记录分析

> **你**：帮我看看这段聊天，我感觉她最近回消息越来越慢了，不知道什么情况

<table><tr><td>
📋 <b>用户上传的聊天记录</b><br><br>
🔵 我：周末去了那家你之前推荐的咖啡馆，拿铁确实不错 ☕<br>
🩷 她：哈哈是吧！我就说那家很好喝<br>
🔵 我：下次可以一起去，我请你<br>
🩷 她：好呀 有机会的<br>
🔵 我：那这周六下午有空吗？<br>
🩷 她：周六可能有事诶 下次吧<br>
🔵 我：好的没事 那你什么时候有空呀<br>
🩷 她：最近挺忙的 到时候再说吧
</td></tr></table>

> **情圣教练**：兄弟，我拆完了，跟你说实话——前半段是有戏的，但后半段你操作上有个关键失误。
>
> **信号拆解：**
> - "哈哈是吧！我就说那家很好喝" → 🟢 **IOI** — 她用了感叹号 + 回旋镖了之前的推荐，热度在线
> - "好呀 有机会的" → 🟡 **模糊窗口** — "好呀"是正面的，但"有机会的"没给具体承诺
> - "周六可能有事诶 下次吧" → 🔴 **IOD 软拒绝** — 拒绝且没有反邀约
> - "最近挺忙的 到时候再说吧" → 🔴 **IOD 二次回避** — 两次拒绝 + 你追问了"什么时候有空"，她感受到的是压迫感
>
> **关键问题**：你从模糊邀约直接跳到追问时间，还追了两次，需求感暴露了。
>
> **行动方案**：
> - **选项 A（推荐）**：冷 1-2 天，然后用一个完全无关的话题重启对话
> - **选项 B**：发一条朋友圈展示你的生活，不找她聊，让她自己来
>
> ⚠ **千万不要再发"那你忙完告诉我"。** 被拒绝两次且没有反邀约 = 暂停邀约，退回升温阶段。

### Demo 2：阶段定位 + 行动方案

> **你**：我跟一个女生认识三周了，朋友介绍的。聊得挺多，她有时候主动找我，也会分享日常。还没见面，但我说想去她推荐的馆子她说好呀。后来就没提了。我想约她但不知道时机对不对。

> **情圣教练**：
>
> ```
> ① 破冰 ✅ → ② 好感 ✅ → ③ 升温 ← 你在这 → ④ 邀约 → ⑤ 约会 → ⑥ 亲密 → ⑦ 确认
> ```
>
> **判断依据**：她会主动找你 + 分享私人生活 = 信任已建立；她说"好呀"给了模糊窗口 = 但还没到正式邀约。你在 **阶段3→阶段4 的过渡节点**。
>
> **行动方案**：
> 1. **今天** — 升温对话，不提见面。"我刚翻到你之前发的那张猫照，突然有个发现——你们俩表情一模一样哈哈 😂"
> 2. **明天** — 在聊天高点自然接回邀约。"对了那家馆子我查了下菜单不错，你周六还是周日方便？"
> 3. **邀约成功后** — 简短确认，主动在高点离场。"那就周六，到时候我去接你。先忙了 💪"
>
> **预判**：
> - 她选了时间 → 进入阶段4，准备约会攻略
> - 她又说"再看看" → 不追问，退回升温，4-5天后再埋线
> - 她主动提时间 → 强 IOI，可以约更有氛围的场地

### Demo 3：冷读 + 朋友圈分析

> **你**：帮我分析这个女生。朋友圈发的是一个人去 798 看莫奈展，配了咖啡和雨天的照片。还转发了村上春树的书评，配文"这段话我读了三遍"。头像是侧脸+耳环的文艺风。签名"慢慢来比较快"。

> **情圣教练**：
>
> | 维度 | 分析 |
> |------|------|
> | **内外向** | 偏内向，享受独处。"一个人去美术馆""坐了一下午"——独处是充电不是孤独 |
> | **审美** | 文艺型，有品味但不装。选莫奈而不是网红展 = 真喜欢 |
> | **社交风格** | 慢热型。"慢慢来比较快"是人生哲学——不喜欢被催被追赶 |
> | **沟通偏好** | 重质量。"读了三遍" = 喜欢有深度的交流，反感肤浅搭讪 |
>
> **开场策略**：她吃的不是搞笑路线，而是"有品味 + 有共鸣 + 不急不躁"。
>
> - **选项 A（推荐）**："798那个莫奈展我也想去，一个人去的体验怎么样？我怕自己看太久被保安催 😂" → 共同兴趣 + 自嘲
> - **选项 B**："感觉你是那种在咖啡馆能坐一下午不看手机的人，猜对了吗？" → 冷读制造好奇
> - **选项 C**："你转的那篇书评，'读了三遍'是哪段？我也喜欢村上，但每次喜欢的段落不一样" → 真实好奇心
>
> ⚠ **千万别这样开场**：❌ "美女你好" ❌ "你朋友圈好文艺哦" ❌ 发一堆表情包

## v3 新特性

- **用户人设系统**：基于你的真实兴趣和特长打造吸引力人设，永远不替你编故事
- **平台差异化**：覆盖微信、探探、陌陌、Soul、Bumble、青藤之恋，针对每个平台给出差异化的 profile 优化和聊天策略
- **截图自动分析**：上传截图直接开始分析，不问多余的问题
- **"不回复"判断**：不是每条消息都要回，告诉你什么时候沉默比说话更有力量
- **快捷指令**：`/regenerate` 重新生成不满意的建议，`/ask` 快速提问模式
- **emoji 融入**：回复建议自然带上表情，可以直接复制粘贴
- **语言人类化**：消除 AI 痕迹，输出像真人聊天一样自然
- **幽默度标尺**：话术标注轻松/调皮/大胆，根据你的性格选择
- **降低攻击性**：温暖幽默型取代毒舌型，调侃让人笑不让人不舒服
- **深度上下文管理**：完整对话时间线、素材存档、引领规划追踪

## v2 特性（保留）

- **多目标上下文管理**：自动识别你在聊谁，为每个女生维护独立档案
- **主动引领模式**：话题引领、情绪引领、节奏引领
- **框架意识进阶**：框架对决、情绪领导力

## 功能概述

覆盖从开场白到确立关系的完整 **七阶段推进体系**：

1. 开场破冰
2. 建立好感
3. 关系升温
4. 邀约见面
5. 约会实战
6. 亲密升级
7. 确立关系

支持上传聊天记录、朋友圈截图、头像、个人档案等素材做深度分析。

- 不知道怎么和女生聊天 / 不会回消息

- 不知道在探探/Bumble/Soul 等平台上怎么优化 profile 和开场白

- 紧急情况需要快速回答（`/ask`）
