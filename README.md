# 情圣 QingSheng — Skill v2

中国男性社交表达与恋爱全流程私人教练。

## v2 新特性

- **多目标上下文管理**：自动识别你在聊谁，为每个女生维护独立档案，跨对话保持连续性
- **主动引领模式**：摆脱一问一答，教你做话题的引领者——话题引领、情绪引领、节奏引领
- **框架意识进阶**：框架对决、情绪领导力、从被动应答到主动掌控
- **更完整的测试覆盖**：10个测试用例覆盖全阶段 + 多人切换 + 引领场景

## 功能概述

覆盖从开场白到确立关系的完整 **七阶段推进体系**：

1. 开场破冰
2. 建立好感
3. 关系升温
4. 邀约见面
5. 约会实战
6. 亲密升级
7. 确立关系

支持上传聊天记录、朋友圈截图、头像、个人档案等素材做深度分析。
