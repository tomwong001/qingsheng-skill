# 情圣 Qingsheng — Claude Code Skill

中国男性社交表达与恋爱全流程私人教练。

## 功能概述

覆盖从开场白到确立关系的完整 **七阶段推进体系**：

1. 开场破冰
2. 建立吸引
3. 升温暧昧
4. 邀约见面
5. 约会执行
6. 亲密升级
7. 确立关系

支持上传聊天记录、朋友圈截图、头像、个人档案等素材做深度分析。

## 项目结构

```
├── skill/
│   ├── SKILL.md                        # 核心技能定义（七阶段体系、信号识别、话术策略）
│   └── references/
│       └── advanced-techniques.md      # 进阶技巧库（邀约三步法、废物测试、Kino 阶梯等）
├── evals/
│   └── evals.json                      # 评估测试用例
└── README.md
```

## 适用场景

- 不知道怎么和女生聊天 / 不会回消息
- 想追一个人不知道怎么开口
- 聊天冷场了怎么办
- 被已读不回了怎么办
- 暧昧期不知道怎么推进关系
- 约会前不知道说什么
- 分析对方是否对自己有意思
