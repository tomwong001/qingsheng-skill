常见问题

10 个用户最常问的问题。

Q: 微信聊天截图怎么用情圣分析？

直接在 Claude Code 里把截图粘贴进去，加一句「帮我看看这段」。情圣会 OCR 读出来、标注谁说的、判断关系阶段、给回复建议。聊天记录越长判断越准。

Q: 探探 / Soul / Bumble 匹配后不知道怎么开场？

把对方的 profile 截图发进来（照片 + bio + 兴趣标签都要），说「帮我想开场白」。情圣会根据她 profile 里的具体信息给 2-3 条开场，不发「在吗」这种低价值开场。每个平台有专门的 playbook。

Q: 女生已读不回怎么办？

别再发「在吗」「你在忙吗」——这种追问会让局势更坏。给情圣看聊天记录，它会判断是真的在忙（正常，等两天）还是被动态度转变（要调整策略）。对第二种有专门的冷激活流程。

Q: 青藤之恋 profile 怎么写更吸引人？

用 /展示面 子命令。情圣会逐项诊断你的头像、bio、兴趣标签、照片组合，给每一项打分 + 具体改法。青藤之恋有专门的严肃交友语气规则，不会让你写成探探风格。

Q: 被女生冷处理后还能挽回吗？

分场景。聊天冷了（< 2 周没回）用 /挽回 的冷激活路径；分手想复合用关系恢复路径。两种都会先做诊断，再给 3-5 步时间线 + 第一条消息。情圣不会鼓励你纠缠——对方明确拒绝时会直接告诉你「到此为止」。

Q: 聊天冷场了怎么重启？

最差的选择就是发「在吗」或「在干嘛」。情圣会根据你们上次聊到哪给出有记忆点的重启话术（比如：之前她提过喜欢一个乐队 → 「那天说到 XX，今天看到他们要开演唱会了」）。有记忆点的重启才不显得功利。

Q: 什么时候该邀约见面？

看阶段。阶段 3（暧昧期）末段 + 聊天里出现地点信号（她提到去过某地 / 想去某地）就是自然的邀约窗口。情圣会在合适的窗口主动提醒你「可以邀约了」，并给你具体的邀约话术（三步法：锚定场景 → 给时间选项 → 具体细节钉死）。

Q: 朋友圈发什么能吸引异性？

不是发豪车豪宅——那是低智商展示。/展示面 会告诉你发什么：真实的生活片段（带情绪 / 带观察角度）、有「勾子」的图文（让人想评论）、节奏（频率不能太高也不能太低）。

Q: 跟 ChatGPT 裸用有啥区别？

① 中文语感 —— ChatGPT 会给「你好，很高兴认识你」这种翻译腔
② 关系记忆 —— ChatGPT 每次对话独立，情圣会把每个妹子的档案存到本地
③ 策略层 —— ChatGPT 只做「这句怎么回」，情圣告诉你该不该回、什么时候回、节奏怎么走

Q: 付费 PUA 课程 vs 情圣 skill？

付费课程教你「装高价值」—— 短期有效，长期崩盘。情圣死守真实性原则：档案里没有的故事不会编，真正靠你自己的特长 + 节奏感 + 对信号的准确判断来吸引人。而且情圣免费，课程动辄 ¥3000-30000。
