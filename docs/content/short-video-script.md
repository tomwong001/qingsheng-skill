## 拍摄/剪辑注意

1. **前 3 秒必须钩住**——第 1 秒就把女生那句"今天加班好累啊"字幕甩出来，让划视频的人停下
2. **对比要狠**：A 方案失败（变灰）→ B 方案成功（秒回 + 聊 20 分钟），给视觉上的反转爽感
3. **字幕密度高**：短视频用户声音经常关着，所有关键信息必须字幕化
4. **Claude Code 终端录屏加速 2x**：让"AI 思考输出"的过程看起来更快更聪明
5. **不要出现真人聊天记录**——全部用演示数据，避免隐私争议
