# 把 Claude Code 改造成了一个中文恋爱教练，开源了，想听听各位的意见

**主打关键词**：Claude Code skill、中文恋爱 AI、微信聊天助手、情圣 skill、开源恋爱教练

---

跟女生聊天不知道怎么回，这个痛点相信不少单身的兄弟都遇到过。

市面上的解法要么是几千块的 PUA 课程（教的东西过时又油腻），要么是直接把聊天记录丢给 ChatGPT（它会一本正经地教你"要真诚要尊重"这种正确的废话）。我自己尝试了半年，发现两条路都走不通——前者价格劝退、后者不懂中文语境下的暧昧分寸。

于是我花了一个多月，把 Claude Code 做成了一个专门的中文恋爱教练 skill，起名叫**情圣（qingsheng）**。今天拿出来分享，顺便听听大家的反馈。

**它能做什么：**

- 读懂微信/探探/Soul/Bumble/青藤之恋/Tinder 的聊天截图（OCR + 中文语义分析）
- 判断女生当前的信号：是在礼貌敷衍，还是真的有兴趣
- 给出 2-3 个不同风格的高情商回复选项，并解释每个选项背后的逻辑
- 追踪多个目标的关系阶段（7 阶段模型：开场 → 建立好感 → 吸引 → 邀约 → 约会 → 升温 → 确立关系）
- 冷场救援、被已读不回的破冰、朋友圈评论策略

**怎么实现的：**

本质上是一个结构化的 SKILL.md（Claude Code 的 skill 加载协议），里面塞了 1400+ 条真实中文对话语料训练出的判断逻辑，加上 prompt 工程把 Claude 的默认"政治正确"倾向压下去。所有计算在本地跑，聊天记录不会上传到任何第三方。

**安装：**

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/tomwong001/qingsheng-skill/main/setup)
```

或者 clone 下来 `./setup`。MIT 协议，代码全开源。
